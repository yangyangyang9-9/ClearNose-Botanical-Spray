# ClearNose Botanical Spray 落地页 PRD

## 1. 产品概述

ClearNose Botanical Spray 是一款植物基鼻腔喷雾产品，旨在通过天然植物成分帮助用户缓解鼻塞、流鼻涕等鼻腔不适，促进呼吸顺畅。本落地页为单一高转化率营销页面，面向全球用户（多语言支持），目标是引导用户完成购买转化并完成 PayPal/Stripe 支付跳转。

- **核心问题**：用户因鼻塞、流鼻涕、夜间呼吸困难而困扰，市面产品多为药物类，缺乏温和的植物基替代方案
- **目标用户**：25-55 岁、关注健康自然疗法、有季节性过敏或鼻腔不适的全球用户
- **市场价值**：通过 SEO 优化 + 高转化落地页设计，实现全球直接销售（DTC 电商模式）

## 4. 用户界面设计

### 4.1 设计风格

- **整体风格**：医疗健康风（Apple Health / Nike 产品页风格），极简设计，非中药风
- **主色**：白色（#FFFFFF）背景为主
- **辅助色**：医疗绿（#10B981 / #059669）+ 信任蓝（#3B82F6 / #2563EB）
- **强调色**：CTA 按钮使用绿色渐变（#10B981 → #059669）
- **文字色**：深灰（#1F2937）主文字 + 中灰（#6B7280）副文字
- **按钮风格**：圆角（rounded-full / rounded-2xl），带阴影和 hover 微动效
- **字体**：标题使用 Plus Jakarta Sans（现代医疗感），正文使用 Inter
- **布局**：单列垂直滚动，卡片式分区，最大宽度容器居中
- **图标风格**：Lucide React 线性图标，圆润现代
- **动效**：滚动渐入动画（fade-in + slide-up），hover 卡片上浮，按钮波纹

### 4.2 页面设计概览

| 区块 | UI 元素 | 设计要点 |

| Footer | 三列布局 + 免责声明 + 链接占位 | 深灰背景 + 白色文字 |

### 4.3 响应式

- **移动端优先**（Mobile First）：默认单列布局，所有区块在手机上垂直堆叠
- **平板**（md, 768px+）：痛点、成分、价格变为 2 列
- **桌面**（lg, 1024px+）：Hero 左右分栏，痛点 2x2，价格 3 列并排
- **触摸优化**：按钮最小点击区域 44x44px，FAQ 折叠全宽可点击

### 4.4 合规要求

- 禁止宣称"治愈疾病"
- 使用合规措辞："helps relieve"（帮助缓解）、"supports nasal comfort"（支持鼻腔舒适）、"promotes easier breathing"（促进更顺畅呼吸）
- Footer 必须包含免责声明："Not a medical product. Not intended to diagnose or treat diseases."
- 用户评价必须标注为 demo/testimonials

## 5. SEO 与分析

### 5.1 SEO 要求

- **Meta Title**：ClearNose Botanical Spray | Natural Nasal Relief & Congestion Support
- **Meta Description**：Breathe easier in minutes with ClearNose Botanical Spray. A gentle plant-based nasal spray to help relieve congestion, runny nose, and seasonal allergies.
- **关键词**：nasal spray, natural nasal relief, congestion relief, botanical spray
- **标题层级**：H1 仅 Hero 标题，H2 各区块标题，H3 子标题
- **Open Graph**：og:title, og:description, og:image 预留
- **结构化数据**：Product schema（价格、评价）
- **快速加载**：图片懒加载 + 代码分割
